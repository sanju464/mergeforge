✨ What is mergeforge?
mergeforge is a fast, privacy-first file conversion and merging tool that runs entirely in your browser. No accounts, no cloud uploads, no waiting — just drop your files and convert.

🚀 Features

🔒 100% Private — all processing happens in your browser. Files are never uploaded anywhere.
⚡ No installation — runs directly in the browser, nothing to download
🔀 Two modes — Convert mode (one file in, different format out) or Merge mode (multiple files in, one combined output)
📦 Wide format support — PDF, PNG, JPEG, WEBP, GIF, SVG, TXT, CSV, JSON, DOCX, PPTX, MP4, MP3
🐳 Docker ready — easy to self-host


📂 Supported Conversions
FromToPDFImages (PNG, JPEG)ImagesPDFImageImage (format change)Any filesZIPTXTPDFCSVJSONCSV / JSONPDFSVGPNG / JPEGDOCXPDFPPTXPDFMP4MP3

⚠️ DOCX, PPTX, and video conversions are not supported client-side and will be flagged as unsupported.


🛠️ How to Use

Open the app at mergeforge.onrender.com
Drop files or click to browse
Select your output format
Choose Convert (single file) or Merge (multiple files)
Download your result instantly

🙋 Why mergeforge?
Most file converters upload your files to a remote server — which is slow, risky, and requires an internet connection. mergeforge does everything locally, making it faster and completely private.
